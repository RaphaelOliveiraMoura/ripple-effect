<div style="background-color: #000; color: #fff; padding: 16px; border-radius: 6px;">

styles.css

`.mycomponent { --wave-color: black; --wave-scale: 10; }`

</div>

<b>--wave-color</b> change the color of ripple

<b>--wave-scale</b> change the size of ripple
